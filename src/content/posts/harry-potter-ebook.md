**ハリーポッター 電子書籍**を **Kindle Unlimited** で読みたい——そう検索する方は多いです。結論から言うと、**各巻・各版ごとに読み放題の対象が異なり、時期によって入ったり抜けたりします**。人気シリーズのため、**「全巻がいつでも追加料金なし」** とは限りません。本記事では巻ごとの断定はせず、**Amazon.co.jp の Kindle 版商品ページで「Kindle Unlimited 読み放題」を確認する手順**と、対象外のときの代替を整理します。料金・対象作品は変更されるため、加入・購入前は公式ページでご確認ください。

## ハリーポッター電子書籍の種類を押さえる

同じ物語でも、Amazon 上では**別商品**として並ぶことがあります。確認のときに混同しないよう、種類を分けておきます。

| 種類 | 確認するときのポイント |
|------|------------------------|
| Kindle 版（電子書籍） | 「Kindle Unlimited 読み放題」の有無を見る |
| 紙の本 | Unlimited の対象外（電子版を別途探す） |
| オーディオブック（朗読） | **Audible** など別商品。読み放題の有無も別途確認 |
| 新訳・旧訳・イラスト版 | **版ごとに商品ページが分かれる** |

読みたいのが「電子で読む」なら、必ず **Kindle 版** のページだけを開いて判断してください。

## 読み放題かどうかの確認方法

『ハリー・ポッターと賢者の石』から最終巻まで、**巻ごと・版ごと**に次の手順を繰り返します。

1. Amazon.co.jp で「ハリーポッター Kindle」または巻名（例：賢者の石 Kindle）で検索する
2. **Kindle 版（電子書籍）** の商品ページを開く
3. 価格付近の **「Kindle Unlimited 読み放題」** の表示があるか見る
4. 表示があれば、その時点で Kindle Unlimited 会員なら、追加料金なしで読み始められることが多い
5. 表示がなければ、その時点では対象外。単品購入・セット・セールを公式で確認する

**無料サンプル（試し読み）** がある版は、翻訳の読みやすさや装丁の好みを確認してから本編に入ると失敗が少ないです。シリーズ物は「第1巻だけ対象、続編は単品」といった**巻単位のばらつき**も起こり得ます。確認の考え方は[『三体』の作品別記事](/posts/sample-ku-title/)と同じです。

## 電子書籍で読むときの実務的なポイント

| 項目 | 内容 |
|------|------|
| 読む順番 | 第1巻から順に読む読者が多い。途中巻だけ対象、もあり得る |
| 端末 | Kindle 端末、Kindle アプリ（[電子書籍タブレットの選び方](/posts/ebook-reading-tablet-guide/)） |
| ハイライト | 気になる箇所をマークして後から見返せる（機能は端末・作品による） |
| シリーズ購入 | セット商品がある場合も。価格は**購入時の表示**を正とする |

長編を続けて読む習慣づけは、[読書習慣が続かない人向けの Kindle Unlimited 活用法](/posts/reading-habit-kindle-unlimited/)も参考になります。

## 対象外のときの代替

| 方法 | 内容 |
|------|------|
| 単品購入 | Kindle 版を都度購入。購入済みは通常ライブラリに残る |
| セット・セール | 公式のセット商品・期間限定価格を商品ページで確認 |
| 第1巻だけ Unlimited | 第1巻を読み放題で読み、続編は都度ページ確認または購入 |
| 別のファンタジー | Unlimited 内の対象作品を[おすすめの探し方](/posts/kindle-unlimited-recommended/)で探す |
| 図書館・紙 | 別ルートでの閲覧 |

**「絶対いま全部読める」** と決めつけず、読み始める直前に商品ページを開き直す習慣がいちばん確実です。

## オーディオブック（聴く）版について

朗読版は **Audible** などの**別商品**です。Kindle 版が読み放題でも、音声版は対象外、という組み合わせがよくあります。聴き放題かどうかは、オーディオブックの商品ページで「含む」等の表示を確認してください。読む・聴くの違いは[Audible と Kindle Unlimited の比較](/posts/ku-vs-audible/)にまとめています。作品別の聴き方の例は[雨穴『変な家』の Audible 確認](/posts/sample-audible-title/)も参照できます。

## シリーズ読みのペースと Kindle Unlimited の料金感

ハリーポッターの全巻を電子で揃える場合、**読み放題に載っている巻だけ Unlimited**、残りは単品、という組み合わせがよくあります。Kindle Unlimited の月額は **980 円（税込）** です。無料体験の条件は申込ページで確認してください。

| 状況 | 考え方 |
|------|--------|
| 第1巻だけ対象 | 第1巻を Unlimited、続きは都度ページ確認 |
| 複数巻が対象 | 月内に読める巻数で 980 円の価値を判断 |
| 全巻対象外 | セット購入・セール・図書館など別ルート |

長編は「1日○ページ」「週末に1巻」など、**小さな目標**にすると挫折しにくいです。読書習慣のコツは[Kindle Unlimited で読書習慣をつける](/posts/reading-habit-kindle-unlimited/)にまとめています。子どもと一緒に読む場合は、版の難易度と試し読みで確認し、必要なら紙や図書館も選択肢に入れてください。

Prime Reading など名称が似たサービスと混同しないよう注意してください（[Prime と Kindle の関係](/posts/kindle-amazon-prime-guide/)）。ハイライトや辞書機能は端末・アプリにより異なります。購入・借り放題の直前に、必ず Kindle 版の商品ページを開き直す習慣をつけると、配信変更によるミスマッチを減らせます。

## よくある誤解

| 勘違い | 実際 |
|--------|------|
| 人気作だから Kindle Unlimited に必ず全部入る | 契約・時期で入れ替わる |
| 映画を見たから電子版も無料 | 動画と電子書籍は別 |
| 1巻だけ Unlimited なら続編も同じ | **巻ごと**にページを確認する |
| 去年のブログのスクショが今も正 | 口コミ・古い記事は現在の保証にならない |
| Prime Reading と Kindle Unlimited は同じ | **別サービス**。混同しない |

## よくある質問

### ハリーポッターは全部 Kindle Unlimited ですか？

**常に全巻対象とは限りません**。各巻の Kindle 版商品ページを確認してください。

### 新訳と旧訳で対象は同じですか？

**版ごとに商品が分かれ、対象も異なる**ことがあります。読みたい版のページだけを見てください。

### 映画公開前後で配信は変わりますか？

カタログは**契約・時期で入れ替わる**ことがあります。古い情報は参考程度にし、都度公式で確認してください。

### ラインナップ一覧だけ見れば十分ですか？

一覧は**候補**です。最終判断は各巻の商品ページ（[ラインナップの見方](/posts/kindle-unlimited-lineup/)）。

### 子どもに読ませたい。年齢や版の選び方は？

内容・難易度は版や翻訳で異なります。商品説明・試し読みで確認し、必要なら紙や図書館も検討してください（本記事は配信確認が主題です）。

## 公式で確認するときのチェックポイント（編集部メモ）

ハリーポッターは版（新訳・イラスト付きなど）が分かれやすく、**検索結果の1件目だけ見て判断しない**ことが重要です。当編集部は、Kindle 版とオーディオブック版が別 SKU であることを前提に記事を構成しています。朗読を検討する場合は Audible 側の商品ページで「含む」等を確認し、読む版とは別契約である点を忘れないでください。Kindle Unlimited の月額・無料体験は [料金・無料体験](/posts/kindle-unlimited-pricing/)の整理を参照しつつ、**申込ページの表示**を最優先してください。セット購入やセール価格も時期により変わるため、購入時の価格表示を正とします。子ども向けに読む場合は、試し読みで難易度を確認し、必要に応じて紙や図書館も選択肢に入れてください。

## 料金と注意（公式情報の整理）

Kindle Unlimited の月額は **980 円（税込）** です。無料体験の有無・期間・終了後の自動課金は、**申込ページの表示のみ**を正としてください。ハリーポッターのように巻数の多いシリーズでは、対象巻が月内に複数あるかどうかで、読み放題の価値が大きく変わります。対象外の巻は単品購入となり、セット価格やセールは**購入時の商品ページ**で確認します。同時に借りられる冊数や借り替えのルールは Amazon の公式ヘルプに従ってください。オーディオブック版は Audible 等の別料金体系であり、読む版が Unlimited でも音声は別契約である点に注意してください。Prime Reading や Prime 特典と Kindle Unlimited は別サービスです。本記事の整理は執筆時点のものであり、配信・料金が変わった場合は Amazon.co.jp の表示を優先してください。

- 人気作は**対象外の期間**もある
- 対象外なら単品購入・セット、または Unlimited 内の別作品から入るのも現実的
- 朗読版は別商品。**最新は Amazon.co.jp の公式表示**を正とする
