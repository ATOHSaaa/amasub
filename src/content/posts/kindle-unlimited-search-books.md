**Kindle Unlimited 使い方**のうち、多くの方が最初に知りたいのが「**対象の本の探し方**」です。**Kindle Unlimited 対象**の本だけを絞り込みたい、**読み放題で読める本の検索方法**を知りたい——そうした検索はとても多いです。結論から言うと、**Kindleストアの検索結果で「Kindle Unlimited」で絞り込む**方法と、**作品ごとのKindle版商品ページで読み放題表示を確認する**方法の2本立てが確実です。アプリの画面は更新されるため、**画面上の文言・ボタンは表示どおり**に操作してください。料金・対象作品は変更されるため、最新は公式ページでご確認ください。

## Kindle Unlimitedの基本的な使い方（流れ）

会員登録後の流れは、端末により画面が異なりますが、次のイメージです。

| ステップ | 内容 |
|----------|------|
| 1. 本を探す | ストア検索・カテゴリ・おすすめ一覧 |
| 2. 読み放題か確認 | Kindle版商品ページの表示 |
| 3. 借りる | アプリで「読む」「借りる」等（表示どおり） |
| 4. 読む | Kindleアプリまたは対応端末で閲覧 |
| 5. 返却・次へ | 同時借り上限がある場合は返却して次を借りる |

料金・無料体験は[Kindle Unlimitedの料金](/posts/kindle-unlimited-pricing/)、解約は[解約方法](/posts/kindle-unlimited-cancel/)を参照してください。初めての方は[おすすめの探し方](/posts/kindle-unlimited-recommended/)でジャンル別の入り口も確認できます。

## 読み放題かどうかの見分け方

Kindle版の商品ページで、価格付近に次のような表示があるか確認します。

- **「Kindle Unlimited 読み放題」**（またはそれに相当するロゴ・文言）

表示がなければ、その時点では**対象外**として、単品購入や別の読み方を検討します。

| 表示 | 意味の目安 |
|------|------------|
| Kindle Unlimited 読み放題あり | 会員なら追加料金なしで読める場合が多い |
| 表示なし | 単品購入が必要なことが多い |
| 無料サンプル | 試し読みのみ。本編は別 |

当編集部が作品記事を書くときも、この表示を**執筆時点で確認**し、変動する旨を明記しています。

## ストアで対象本だけを探す（使い方）

### Web（Amazon.co.jp）

1. Kindleストアでキーワード（著者名・シリーズ名など）を検索する
2. 検索結果やフィルターで **Kindle Unlimited**（読み放題）に絞り込む（表示がある場合）
3. 候補の本を開き、上記の読み放題表示を**再確認**する
4. 気になる本は**無料サンプル**で続きが読みたいか試す

### Kindleアプリ

1. アプリ内の検索窓にタイトル・著者を入力する
2. 結果一覧で **Kindle Unlimited** のタブやフィルターがあれば選択する
3. 商品ページで読み放題表示を確認する
4. ダウンロード（借りる）して読み始める

「検索結果に出た＝ずっと読み放題」ではなく、**カタログは入れ替わる**ため、読み始める直前にもページを開き直す習慣が安全です。

## 作品名が決まっているとき（最短ルート）

読みたい1冊が決まっている場合は、次の順が最短です。

1. 「作品名 Kindle」で検索する
2. **Kindle版（電子書籍）** を選ぶ（紙・オーディオブックと混同しない）
3. 読み放題表示を確認する
4. 対象外なら、[おすすめの探し方](/posts/kindle-unlimited-recommended/)で代替を探すか、単品購入を検討する

作品別の例:[『三体』](/posts/sample-ku-title/)、[『弱虫ペダル』](/posts/yowamushi-pedal-free/)。対象外が多いと感じたら[やめとけ・おすすめしない理由](/posts/kindle-unlimited-not-recommended/)も参照してください。

## ジャンル別の探し方のヒント

| ジャンル | 探し方のコツ |
|----------|-------------|
| 小説 | 作家名・シリーズ名で検索し、巻ごとに確認 |
| 漫画 | 完結・バックナンバーは対象になりやすい。最新話は要注意 |
| ビジネス書 | ジャンル絞り込み＋話題書は単品のみも多い |
| 雑誌 | 号・ムック単位で確認（[雑誌・新聞の整理](/posts/kindle-unlimited-newspaper/)） |

漫画の比較は[漫画サブスク比較](/posts/manga-subscription-comparison/)、ビジネス書は[ビジネス書の活用法](/posts/kindle-unlimited-business-books/)を参照してください。

## 同時に借りられる冊数・返却

読み放題には**同時に借りられる冊数の上限**がある場合があります。詳細はAmazonのヘルプを参照し、記事では数値を断定しません。読み終えたら次の本を借りる、というサイクルで使う読者が多いです。

| 状況 | 対処の考え方 |
|------|-------------|
| 上限に達した | 読み終えた本を返却してから次を借りる |
| 途中でやめたい | 返却して別の本を試す |
| 解約した | 借りていた本は読めなくなる（購入本は別） |

## お気に入り・リストで探しを効率化する

検索のたびに同じキーワードを打つより、**候補をリスト化**しておくと Kindle Unlimited の使い方が安定します。

| 方法 | 使い方 |
|------|--------|
| 欲しい本リスト | 読みたい本を先に登録し、対象になったら借りる |
| シリーズ単位 | 第1巻だけ対象確認し、続巻も巻ごとにページ確認 |
| 著者名検索 | 作家の他作品も同じ手順で読み放題表示を見る |

リストに入っていても、**借りる直前に商品ページを開き直す**習慣は外さないでください。カタログ入れ替えで対象外になることがあります。

## 複数端末で読むときの同期

KindleアプリとKindle端末を併用する場合、同じAmazonアカウントでログインすれば、読書位置の同期が効くことが多いです。ただし次の点は端末・OSで挙動が異なることがあります。

| 項目 | 補足 |
|------|------|
| ダウンロード済みの本 | 端末ごとに再ダウンロードが必要な場合がある |
| フォント・余白 | 端末ごとに設定が別になることがある |
| オフライン | 出かける前に借りた本をダウンロードしておく |

端末選びは[電子書籍タブレットの選び方](/posts/ebook-reading-tablet-guide/)も参考にしてください。

## 検索で見つからない・ヒットが少ないとき

| 原因 | 対処 |
|------|------|
| 紙のタイトル名と電子版名が違う | 「Kindle」「電子書籍」で再検索 |
| オーディオブックで検索している | **Kindle版**に切り替える |
| 配信終了 | 別タイトル・別出版社版を探す |
| 絞り込みが強すぎる | フィルターを外してから再検索 |

それでも対象がなければ、[読める本が少ないと感じるとき](/posts/kindle-unlimited-few-books/)の整理や、[やめとけ・おすすめしない理由](/posts/kindle-unlimited-not-recommended/)で見送り判断を検討してもよいです。

## 借り方・返却の操作イメージ（画面は更新されます）

1. Kindle版商品ページで読み放題表示を確認する
2. **読む**または**借りる**（表示どおり）をタップする
3. ライブラリに追加されたことを確認する
4. 読み終えたら、必要に応じて返却する（上限対策）
5. 次の本を同じ手順で借りる

ボタン名はアプリのバージョンで変わるため、**画面上の文言**に従ってください。解約後の挙動は[解約方法](/posts/kindle-unlimited-cancel/)と公式ヘルプを参照してください。

## よくある誤解

| 誤解 | 実際 |
|------|------|
| Unlimited会員なら全検索結果が読める | 対象作品のみ |
| 一度借りたら永久に読める | 解約・返却ルールは公式ヘルプを参照 |
| 紙の本のISBNで同じタイトルが必ず対象 | 版・出版社・電子版ごとに異なる |
| プライム会員なら同じ | Prime Reading とは別カタログ |

[本のサブスク・電子書籍サブスク](/posts/ebook-subscription-guide/)でも、サブスク全般の誤解を整理しています。

## よくある質問

### Kindle Unlimitedの本だけ検索できますか？

ストアやアプリの**絞り込み機能**で可能な場合があります。機能名は更新されるため、画面の表示に従ってください。

### 漫画も同じ手順ですか？

はい。ただし**最新話が対象外**のことが多いです。[漫画サブスク比較](/posts/manga-subscription-comparison/)を参照してください。

### 解約すると検索履歴は消えますか？

アカウント・端末の挙動は公式の説明に従ってください。解約手順は[解約方法](/posts/kindle-unlimited-cancel/)。

### 料金を知りたい

[Kindle Unlimitedの料金・無料体験](/posts/kindle-unlimited-pricing/)を参照してください。

### 聴く版（オーディオブック）は同じ手順ですか？

**Audible** は別サービスです。[AudibleとKindle Unlimitedの違い](/posts/ku-vs-audible/)を参照してください。

### プライム会員でも検索の仕方は同じですか？

Kindle Unlimited 会員であれば、**絞り込みと商品ページの読み放題表示**の確認手順は同じです。Prime Reading だけ使う場合は別カタログです（[プライムリーディングとは](/posts/prime-reading-guide/)）。

### 無料サンプルと読み放題の違いは？

無料サンプルは**試し読み**です。本編全体が読み放題かは、別途「Kindle Unlimited 読み放題」の表示があるかで判断します。

### 借りた本を家族と共有できますか？

アカウント・家族共有のルールは**Amazonの利用規約とヘルプ**に従ってください。記事では共有可否を断定しません。

## まとめ

- 対象本は**ストアの絞り込み＋商品ページの読み放題表示**で確認する
- 作品名が決まっているときは**Kindle版のページ**だけを見る
- カタログは変わるため、読み始める前に**再確認**する
- おすすめの探し方は[別記事](/posts/kindle-unlimited-recommended/)、料金は[料金記事](/posts/kindle-unlimited-pricing/)へ

料金・キャンペーン・対象作品は変更されることがあります。最新情報は各サービスの公式ページでご確認ください。
