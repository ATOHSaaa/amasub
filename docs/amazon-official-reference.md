# Amazon 公式情報リファレンス（アマサブ）

Kindle Unlimited・Audible に関する**料金・機能・公式URL・確認手順**を記事に書くときは、本ファイルを正とする。

- 競合サイトや古い記事の数値は**そのまま使わない**
- 本ファイルと公式ページが食い違う場合は**公式ページを優先**し、本ファイルを更新する
- 作品ごとの読み放題/聴き放題の可否は**常に変動**する。作品の可否は商品ページで確認する旨を記事に書く

**最終確認日（編集部）:** 2026-06-01  
（料金改定時はこの日付と各セクションを更新すること）

---

## 1. 共通

### 公式ドメイン

- 日本: `https://www.amazon.co.jp/`
- 記事内リンクは可能な限り **amazon.co.jp** の公式URLを使う

### 記事に必ず入れる注意（定型）

> 料金・キャンペーン・対象作品は変更されることがあります。最新情報は各サービスの公式ページでご確認ください。

### アフィリエイト

- 当サイトは Amazon.co.jp アソシエイト・プログラムに参加している
- **トラッキングID:** `amasubweb-22`（環境変数 `PUBLIC_AMAZON_ASSOCIATE_TAG`）
- 記事末尾の CTA・開示文はサイト側で自動挿入される（本文に長文の開示を重複させない）

### コードとの対応

サイト内 CTA 等で使う URL は `src/config/site.ts` の `services` と揃える。本ファイルを更新したら、必要に応じて `site.ts` も更新する。

## 2. Kindle Unlimited

### 公式ページ（リンク先）

| 用途 | URL |
|------|-----|
| サービス概要・申込 | https://www.amazon.co.jp/kindle-dbs/hz/signup |
| Kindle ストア（トップ） | https://www.amazon.co.jp/kindle-dbs/store |

### 料金（執筆時の参照値）

| 項目 | 内容 |
|------|------|
| 月額 | **980円（税込）** |
| 無料体験 | 期間・条件はキャンペーンにより変動。**申込ページの表示を正とする** |

※ プライム会員向けの特典・学割等はある場合がある。記事で触れる場合は「公式の料金ページで確認」とし、断定を避ける。

### サービス内容（公式ベースの説明）

- 対象の電子書籍・漫画・雑誌などを、対応デバイスで**読み放題**（借り放題）
- 対象作品数は Amazon の表記に従う（「500万冊以上」等。**数字は公式表示を確認**してから記載）
- 1冊あたりの同時貸出冊数には上限がある（詳細は公式ヘルプを参照）
- **すべての Kindle 本が対象ではない**

### 読み放題かどうかの確認手順（記事用）

読者向けに、次の手順を記事に載せる:

1. Amazon.co.jp で対象の本を検索する
2. **電子書籍（Kindle版）** の商品ページを開く
3. 価格付近に **「Kindle Unlimited 読み放題」**（またはそれに相当するロゴ・文言）があるか確認する
4. ロゴがない場合、その時点では Unlimited 対象外として単品購入等が必要

### 記事で使ってよい表現 / 避ける表現

| 使ってよい | 避ける |
|-----------|--------|
| 「Kindle Unlimited 対象の場合、追加料金なしで読める場合があります」 | 「必ず読み放題」「全巻無料」 |
| 「配信状況は時期により変わります」 | 過去のブログのスクショのみを根拠にした断定 |
| 「公式の商品ページでご確認ください」 | 非公式まとめサイトの料金のみを根拠にする |

### よくある読者の誤解（記事で補足するとよい）

- **単品購入した本**と Unlimited 対象は別（購入済み本はライブラリに残るが、対象外作品を Unlimited で読むには対象である必要がある）
- **Audible オーディオブック**とは別サービス
- 紙の本・Prime Reading 等とは別のサービス（混同しない）

## 3. Audible

### 公式ページ（リンク先）

| 用途 | URL |
|------|-----|
| サービス概要・申込 | https://www.amazon.co.jp/hz/audible/mlp |
| Audible トップ | https://www.amazon.co.jp/Audible |

### 料金（執筆時の参照値）

| 項目 | 内容 |
|------|------|
| 月額 | **1,500円（税込）**（プランにより異なる場合あり） |
| 無料体験 | 期間・条件は変動。**申込ページの表示を正とする** |

※ 「Audible会員」「Audible Premium Plus」等、プラン名称は変更されることがある。記事では**現在の公式名称**を確認してから使う。

### サービス内容（公式ベースの説明）

- **オーディオブック**（朗読・音声コンテンツ）の聴取サービス
- 対象作品はタイトルごとに異なる（**聴き放題プランの対象**か、単品購入かを商品ページで確認）
- スマホアプリ「Audible」での聴取が中心
- **すべての本・すべての朗読が聴き放題とは限らない**

### 聴き放題かどうかの確認手順（記事用）

1. Amazon.co.jp または Audible で対象作品を検索する
2. **オーディオブック版** の商品ページを開く
3. **「含む」「Audible会員なら聴き放題」** 等の表示があるか、またはサンプル再生で確認する
4. 表示がない・購入のみの場合は、プラン対象外または別途購入が必要

### 記事で使ってよい表現 / 避ける表現

| 使ってよい | 避ける |
|-----------|--------|
| 「Audible で配信されている場合」「会員プランの対象なら」 | 「全部聴き放題」「必ず無料」 |
| 「ナレーション・収録版によって体験が異なります」 | Kindle 版の有無と混同した説明 |
| 「公式ページで最新の配信状況を確認」 | 未検証の口コミのみを根拠にする |

### Kindle Unlimited との関係（比較記事用）

| 観点 | Kindle Unlimited | Audible |
|------|------------------|---------|
| 主な体験 | 読む（電子書籍） | 聴く（オーディオブック） |
| 月額の目安 | 980円 | 1,500円 |
| 向いている場面 | じっくり読書、漫画・雑誌 | 通勤・家事中のながら聴き |
| 作品の重複 | 一部タイトルは両方あるが**別SKU・別契約が多い** | 同上 |

両方に加入して使い分ける読者もいる。比較記事では**排他的にどちらか一方だけ**と決めつけない。

### 記事で押さえるポイント

- **手書き入力**は **Kindle Scribe 系3モデルのみ**（比較表の「手書き入力機能」行）。
- **防水**は比較表では **Paperwhite / Colorsoft（7インチ）** にあり、**Scribe 系には記載なし**。Scribe記事で「防水」と書かない。
- **画面サイズ**: 読み専用の定番は **6インチ（Kindle）** と **7インチ（Paperwhite / Colorsoft）**。Scribe は **11インチ**。
- **カラー読書**を主目的にするなら **Kindle Colorsoft**（7インチ）または **Kindle Scribe Colorsoft**（11インチ・手書きも可）。いずれも電子ペーパー方式で、タブレット型LCDとは異なる。
- 旧世代（例: 2022年・2024年発売の10.2インチ Scribe）は比較表の現行8モデルに含まれない場合がある。**現行比較表と商品ページ**を優先し、旧モデルは「在庫・中古で残る場合あり」程度に留める。
- 端末価格を記事本文に書く場合は **「参考価格（〇〇年〇月時点）」** とし、Creators API カードまたは公式ページへの誘導を併記する。

### 記事で使ってよい表現 / 避ける表現

| 使ってよい | 避ける |
|-----------|--------|
| 「公式の比較表・商品ページで最新モデルを確認」 | 旧ブログの型番・価格のまま断定 |
| 「Scribe は読む＋書く向け、Paperwhite は携帯性重視」 | 全Kindleに手書き・防水があるような書き方 |
| 「Colorsoft はカラー表示対応のKindle」 | Scribe と Colorsoft の混同（別製品） |

## 5. 記事タイプ別：公式情報の載せ方

### 作品別（読める？聴ける？）

必須:

- 確認手順（上記 §2 または §3）
- 変動する旨の注意
- 対象外のときの代替（単品購入、サンプル、関連作品）

任意:

- 作品のあらすじ・魅力（著作権・事実に留意。長い引用は避ける）

### 比較・活用法

必須:

- 料金は本ファイルの参照値＋「公式で確認」
- 公式申込ページへの誘導（CTA は自動挿入可）

### キャンペーン・無料体験

- **終了日・条件は必ず公式で確認**
- 「〇月〇日まで」等の日付は、執筆・公開時点で公式を見てから書く。古い記事の日付を流用しない

## 6. 更新ルール（編集部）

次のときは本ファイルを更新し、必要なら関連記事の `updatedDate` を見直す:

- Amazon が Kindle 端末のラインナップ・価格・型番を変更したとき
- Amazon が料金・プラン名を変更したとき
- 無料体験の条件が変わったとき
- 公式 URL がリダイレクト・変更されたとき
- 対象冊数・同時貸出などのルールが公式ヘルプで改定されたとき

更新後:

1. 本ファイル先頭の「最終確認日」を更新
2. `src/config/site.ts` の `description` / `officialUrl` を確認
3. 古い料金が固定された記事があればリライトを検討
